== Description ==

Add a WhatsApp message button to your website and allow your visitors to send you a message with the tap of a button.

=== Features ===

* Add a WhatsApp button to a post or page from directly within the editor.
* Customize the button text and background color.
* Set a default first message to start new chats with.
